# Agentic Launch Kit

A polished Next.js starter tailored for rapid, agentic product launches on Vercel. It ships with a conversion-focused landing page, reusable sections, and zero-config deployment scripts.

## Getting Started

Install dependencies and run the local dev server:

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm start
```

## Deploy

This project is optimized for Vercel. Deploy with:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-ad666b5a
```
